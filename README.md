# Overview

Odysseus is a Python class designed to facilitate querying trail availability at national parks. It leverages external APIs to fetch and analyze data regarding trail reservations and displays availability for the next few months. This class is ideal for outdoor enthusiasts, hikers, and anyone interested in exploring national parks while needing to plan around available dates.

# Features

- Fetch availability data from an external API for specific park IDs.
- Filter days with a minimum number of available spots.
- Calculate availability for the next 'n' months, providing flexibility in planning.
- Customizable output via a user-defined function, allowing integration with different user interfaces or logging systems.
